+ 내 컴퓨터에 있는 git 저장소를 새로 만든 github 저장소와 연결
+ 여러분은 주소 부분에 여러분의 github 저장소 주소를 넣어야 합니다
```bash
git remote add origin https://github.com/kihoon1218/ex01.git
git branch -M main
git push -u origin main
```

+ 내 git 저장소에 파일 변경사항이 발생
  + add
  + commit

  + 내 git 저장소의 커밋 내역을 github 저장소에 올리기
    + ```bash
    git push -u origin main
    ```
